# 100 Days of Code: The Complete Python Pro Bootcamp

Homework repository for the [100 Days of Code™: The Complete Python Pro Bootcamp](https://www.udemy.com/course/100-days-of-code/) (Udemy).

## Structure

- `course/` — materials downloaded from the course
- `projects/` — self-completed coding exercises, named `NNN[L]_<exercise_name>/`
- `resources/` — additional reference materials

## Setup

Requires [uv](https://docs.astral.sh/uv/) and Python 3.14.

```bash
uv run projects/001b_band_name_generator/001_main.py
```
